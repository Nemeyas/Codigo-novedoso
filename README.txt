Actividad 2: Analisis de complejidad temporal

En este codigo se repite un bucle de manera indefinida hasta que el usuario decidad terminar el procesom. Sin embargo las operaciones internas del algoritmo solo se basan en el analisis de distintos casos de manera ramificada
por lo tanto la complejidad de este algoritmo es de O(n) con n siendo la cantidad de iteraciones que se realicen hasta que el usuario cierre el programa.
